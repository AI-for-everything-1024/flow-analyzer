# 数据质量监控与审计

确保流量数据的准确性、完整性和一致性。

## 1. 完整性稽核 (Completeness)
*   **指标对比**: 定期对比 Kafka 摄取条数 vs Doris 写入条数。
*   **告警阈值**: 丢包率 > 0.1% 时触发报警。

## 2. 准确性验证 (Accuracy)
*   **非法值检查**: 监控 IP 字段是否包含非法格式、Port 是否超出 65535。
*   **归一化质量**: 检查 `FlowRecord` 中的 `protocolType` 是否均为已知类型。

## 3. 成本治理 (Cost Governance)
*   **膨胀率监控**: 定期检查 Doris 的压缩比。
*   **Top-N 占用**: 识别存储占用最大的表和分区，评估是否需要缩短保留周期。

## 4. 数据安全审计
*   **访问日志**: 记录所有对 DWD 明细层的高频下载操作。
*   **脱敏校验**: 验证导出任务是否严格执行了 IP 掩码处理逻辑。
